# Convert HEIC files to JPG

Convert your `.heic` files to `.jpg`.

## Running the app

Clone repo and
```
yarn install
yarn start
```

## The converter
The underlying package that does the converting is [heic2any](https://github.com/alexcorvi/heic2any).
